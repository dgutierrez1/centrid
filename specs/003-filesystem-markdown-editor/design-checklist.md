## API Integration

### Document Operations
- [ ] `GET /api/documents` - Fetch file tree
- [ ] `POST /api/documents` - Create new document

- [ ] `DELETE /api/documents/:id` - Delete document
- [ ] `POST /api/documents/:id/rename` - Rename document
